# Toggle App

A simple React Toggle App built using React Hooks (`useState`).

## Features

- Toggle ON/OFF state
- React Functional Components
- useState Hook
- Beginner Friendly UI

## Tech Stack

- React JS
- JavaScript
- CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/ThulaxanUthayakumar/Toggle-App.git
````

Go to project folder:

```bash
cd Toggle-App
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```
